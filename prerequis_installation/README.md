## Prerequisites : 

Two essential tools to create and manage virtual machines:
- **SSH** 
- **Add sudo user**


## 1. SSH Setup
To connect from host to virtual machine debian. 
launch on your virtual debian machine : prerequis_installation/to_insall_ssh.sh

## 2. Add sudo user
To create a user with sudo privileges. It will be used to run the commands without 'sudo' each time.
launch on your virtual debian machine : prerequis_installation/add_sudo_user.sh
